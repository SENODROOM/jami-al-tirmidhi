# Changelog

## [1.0.0] — 2026-03-21

### Added
- Python package published to PyPI (`pip install jami-al-tirmidhi`)
- Python API identical to npm — same camelCase method names
- Monorepo structure — single `data/tirmidhi.json.gz` shared by JS and Python
- `data/`, `src/`, `types/`, `scripts/`, `python/`, `examples/`, `docs/`, `tests/`
- `.github/workflows/` — auto-publish to npm and PyPI on GitHub release
- Offline-first — data bundled, no CDN required
