# Contributing

## Getting Started

```bash
git clone https://github.com/SENODROOM/jami-al-tirmidhi.git
cd jami-al-tirmidhi
```

## Making Changes

- JS changes → edit files in `src/`
- Python changes → edit files in `python/jami_al_tirmidhi/`
- Data changes → edit `data/tirmidhi.json` then run `node scripts/build.mjs`

## Publishing

Publishing is automatic via GitHub Actions on every GitHub Release.
