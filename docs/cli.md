# CLI Reference — jami-al-tirmidhi

```
tirmidhi <hadithId>
tirmidhi <chapterId> <hadithId>
tirmidhi --search "<query>" [--all]
tirmidhi --chapter <id>
tirmidhi --random
tirmidhi --react
tirmidhi --version
tirmidhi --help
```

## Language flags
- (default) — English only
- `-a` / `--arabic` — Arabic only
- `-b` / `--both` — Arabic + English
