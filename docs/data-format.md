# Data Format — tirmidhi.json

```json
{
  "metadata": {
    "id": 1, "length": 3956,
    "arabic":  { "title": "جامع الترمذي", "author": "الإمام أبو عيسى محمد الترمذي" },
    "english": { "title": "Jami al-Tirmidhi", "author": "Imam Abu Isa Muhammad al-Tirmidhi" }
  },
  "chapters": [ { "id": 1, "arabic": "...", "english": "Purification" } ],
  "hadiths": [
    { "id": 1, "chapterId": 1, "arabic": "...", "english": { "narrator": "...", "text": "..." } }
  ]
}
```

`data/chapters/` is generated by `node scripts/build.mjs`. Do not edit manually.
